# Ginkgo AI — The Advice Engine (intro page)

An internal landing/portal page for Ginkgo Financial advisers and AI Pathfinders
partners. It introduces the Advice Engine and links through to the app.

- **The app it links to:** https://ginkgo-advice-demo.streamlit.app
- **Audience:** internal advisers only — not for client or public distribution
- **Status:** prototype; dummy/anonymised data only; every output needs human sign-off

## Files
- `index.html` — the whole website (one self-contained page)
- `assets/` — logo and brand graphics
  - `GF_Landscape_RGB_OnWhite.png` — header logo (already included)
  - optional extras listed in `assets/PUT-BRAND-FILES-HERE.txt`

## To view it
Double-click `index.html` to open it in your web browser.

## To publish it (GitHub Pages)
See the step-by-step guide your assistant provided, or the short version:
1. Create a new repository on github.com and upload `index.html` and the `assets` folder.
2. Repository **Settings → Pages → Deploy from branch → main / root → Save**.
3. Your page goes live at `https://<your-username>.github.io/<repo-name>/`.
