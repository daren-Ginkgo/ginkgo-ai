BRAND FILES FOR THE WEBSITE
===========================

so the header looks right out of the box.

The site will also use these OPTIONAL extras if you drop them in here, using the
exact names below (copy from the SharePoint brand bible / asset package). Without
them the page still looks fine - they just add a little polish.

HOW TO ADD THEM
---------------
Drag-and-drop each file into this "assets" folder, keep the names the same,
then refresh the page. That's it.
